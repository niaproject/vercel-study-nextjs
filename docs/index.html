    Hello Vercel

    Next.js + Vercel デプロイ学習用のサンプルアプリです。

    技術スタック

    • Next.js 16 (App Router)
    • React 19
    • Tailwind CSS v4
    • TypeScript

    ローカル開発

    npm install
    npm run dev

    http://localhost:3000 をブラウザで開いてください。

    ビルド

    npm run build
    npm start

    Vercel へのデプロイ

    1. このリポジトリを GitHub にプッシュ
    2. Vercel でリポジトリをインポート
    3. 設定はデフォルトのまま「Deploy」をクリック

    詳細は Next.js デプロイドキュメント を参照してください。
